How make to publish page on github

1 repo creat

2 cmd > cd path

3 cmd > git clone git_repo_url

(This command will created new folder in to have second step have location.This folder name is same with repo name.This mean new project path is path+reponame)

4 cmd > cd path/reponame (We should see *.git this location)

5 cmd > git init (for sync)

6 cmd > code . (for open vs code with empty project)

7 creat intex.html(this name so important for publish page.WebService is search to firstly this item)

8 (! + enter ) + type some contets to index.html & cntrl+s

9 git add .

10 git commit -m "xcommitx"

11 git push

12 go to git web site & open repo >setting >pages

13 Branch >(main/master) select and save

14 Git should give to one url and publish to intex.html page this url after some time like this sample "https://bipolat.github.io/publish/" ;)
